# Lecture Notes Generator

Веб-приложение для автоматического создания конспектов лекций в формате PDF на основе видеозаписей.  
Пользователь передаёт публичную ссылку на видеофайл, размещённый в Яндекс Диске, после чего система асинхронно обрабатывает материал и формирует готовый конспект.

Проект реализован с использованием **serverless-архитектуры** в **Yandex Cloud** и полностью разворачивается через **Terraform**.

---

## Назначение проекта

Приложение автоматизирует процесс подготовки учебных конспектов и выполняет следующие действия:

1. Приём задания от пользователя через веб-интерфейс  
2. Проверка корректности публичной ссылки Яндекс Диска  
3. Асинхронная обработка видеофайла  
4. Распознавание речи  
5. Формирование структурированного текста лекции  
6. Генерация PDF-документа  
7. Хранение результатов в Object Storage  

---

## Архитектура решения

Проект разделён на два независимых serverless-компонента.

### Web-приложение (Serverless Container — App)

Функции:
- отображение пользовательского интерфейса;
- приём данных от пользователя;
- валидация входных параметров;
- создание и сохранение задания;
- отправка ID задания в очередь сообщений;
- отображение списка заданий и их статусов.

Интерфейс реализован с использованием HTML-шаблонов и работает через HTTP.

---

### Фоновый обработчик (Serverless Container — Worker)

Функции:
- получение заданий из очереди сообщений;
- загрузка видеофайла;
- извлечение аудио с помощью `ffmpeg`;
- отправка аудио в Yandex SpeechKit для распознавания речи;
- генерация конспекта лекции через YandexGPT;
- создание PDF-документа;
- загрузка результата в Object Storage;
- обновление статуса задания.

Обработка выполняется асинхронно и не блокирует работу веб-приложения.
